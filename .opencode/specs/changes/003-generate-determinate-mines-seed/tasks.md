# План задач: Детерминированная генерация поля по seed

## Общая информация
- **Статус**: Спецификация утверждена
- **Формат roomId**: Base62, 10-12 символов
- **Алгоритм**: cyrb128 + sfc32
- **Архитектура**: Параметр `random?: () => number` в `generateMines` (3-й параметр)
- **Общее время**: ~9.5 часов

---

### Критерии приемки
- [ ] `generateMines` работает без 3-го параметра (использует Math.random)
- [ ] `generateMines` работает с кастомным генератором
- [ ] `createSeededRandom` создает детерминированный генератор
