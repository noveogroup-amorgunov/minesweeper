# Сохранение и загрузка игры из файла (OPFS)

В рамках данной апичи нужно реализовать возможность сохранять и загружать игру из файла, используя браузер на API OPFS (Origin private file system).

Для реализации этого в файле `src/engine/GameEngine.tsx` нужно добавить два метода: `load` и `save`, которые будут сервизировать и десервизировать состояние игры, а также загружать файл в PFS.

Также нужно добавить в `src/view/game/GameView.tsx` слой выпадающее меню с возможностью сохранить игру и загрузить игру.

Для простоты можно создавать один единственный файл, в рамках которого мы сохраняем текущую игру и загружаем игру, то есть без конкретного имени игры, которое нужно загружать либо сохранять.
